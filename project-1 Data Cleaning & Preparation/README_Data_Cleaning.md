# Ecommerce Data Cleaning using Excel

## Project Overview
This project focuses on cleaning and preparing an ecommerce sales dataset using Microsoft Excel. The goal was to make the data accurate, organized, and ready for analysis and reporting.

## Dataset Summary
- **Rows:** 1,200
- **Columns:** 16
- **File Type:** Excel Workbook (.xlsx)

## Tools Used
- Microsoft Excel
- GitHub

## Data Cleaning Performed
The following cleaning steps were applied to improve the quality of the dataset:

- Removed or handled missing values
- Standardized column values
- Checked for duplicate records
- Formatted the date column
- Added new analytical columns for better insights

## New Columns Added
To make the dataset more useful for analysis, the following columns were added:

- **Month** – extracted from the order date
- **SalesCategory** – categorized total sales into performance groups

## Key Columns in the Dataset
- OrderID
- Date
- CustomerID
- Product
- Quantity
- UnitPrice
- ShippingAddress
- PaymentMethod
- OrderStatus
- TrackingNumber
- ItemsInCart
- CouponCode
- ReferralSource
- TotalPrice
- Month
- SalesCategory

## Project Objectives
- Clean raw ecommerce data in Excel
- Organize the dataset for analysis
- Create useful helper columns for reporting
- Prepare the file for GitHub portfolio sharing

## Possible Insights from the Dataset
This cleaned dataset can be used to analyze:
- Monthly sales trends
- Product demand
- Payment method preference
- Order status distribution
- Coupon usage
- Customer behavior
